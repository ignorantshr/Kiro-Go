# Triage Labels

Triage label mapping is not configured for this repo.

This repo does not currently use an issue-tracker-based triage workflow, so the canonical mattpocock/skills roles are not mapped:

- `needs-triage`
- `needs-info`
- `ready-for-agent`
- `ready-for-human`
- `wontfix`

## Implications for skills

Skills should not create, apply, remove, or rely on triage labels unless the user explicitly defines a tracker workflow and label vocabulary for the current task.

If issue-tracker triage is introduced later, replace this file with a concrete mapping table.
